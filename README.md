# Phoo Wai Thaw — Quantity Surveyor Portfolio

Static HTML/CSS/JS portfolio. No build step.

## Files
- `index.html`
- `styles.css`
- `script.js`

## Deploy to GitHub Pages
1. Create a new repo, e.g. `phoowaithaw.github.io` (use your GitHub username for a user site).
2. Upload these 3 files to the repo root.
3. Go to **Settings → Pages → Source = `main` / root**.
4. Your site will be live at `https://<your-username>.github.io/`.

## Customize
- Edit text directly in `index.html`.
- Change accent color via the `--accent` CSS variable in `styles.css`.
- Update stats in the `data-target` attributes inside the About section.
- Add/remove project cards inside `<section id="projects">`.
